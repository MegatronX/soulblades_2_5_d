# RiverPathChannel Guide

Use `res://assets/scenes/exploration/RiverPathAuthoringSandboxScene.tscn` to author curved rivers.

Core node:
- `res://scripts/exploration/props/RiverPathChannel.cs` (attached to a `Path3D`)

## Basic workflow

1. Open `RiverPathAuthoringSandboxScene.tscn`.
2. Select `RiverPathChannel`.
3. Edit the `Curve` points directly in the 3D editor to draw the river path.
4. Tune cross-section values in Inspector:
- `CrossSectionStyle`
- `WaterWidth`, `WaterSurfaceYOffset`, `BedDepth`
- `SmoothBankRun` or `GorgeRimHeight`/`GorgeLedgeRun`
5. Tune flow behavior:
- `FlowMode` (`AlongPathForward`/`AlongPathBackward`/`CustomVector`)
6. Tune obstacle and boat helpers:
- `GenerateObstacleAnchors`, `ObstacleAnchorInterval`, `ObstacleEdgeBias`
- `GenerateBoatPath`, `BoatPathYOffset`, `EnableBoatPreview`

## Embedding into real maps

1. Instance `RiverPathChannel` into your exploration map scene.
2. Reuse your tuned materials:
- `WaterMaterial`
- `TerrainMaterial`
- `BedMaterial`
- `GorgeWallMaterial`
3. Keep river corridor and nearby terrain authored as separate geometry layers:
- river channel generated by `RiverPathChannel`
- surrounding map terrain as independent geometry/sprite terrain kits
4. Place obstacle props by snapping to generated obstacle anchors, then convert to final prop nodes.
5. For boat gameplay, use generated `__Generated/BoatPath` as motion path for boat actors.
